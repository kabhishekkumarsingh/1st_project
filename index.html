The Basic Language of the Web:HTML

Posted by laura-joneson Monday,June 21st 2017

All modern websites and web applications are built using three fundamental technologies: HTML,css and JavaScript. These are the languages of the web.

In this post, let's focus on HTML. we will learn what HTML is all about, and why you too should learn is.

What is HTML?

HTML stands for HyperText MarkupLanguage. It's a markup language that Web developers use to structure and describe the content of a webpage(not a programming language)

HTML consists of elements that describe different types of content: paragraphs, links,headings images,video,etc. Web browsers understand HTML and render HTML code as wevsite.
